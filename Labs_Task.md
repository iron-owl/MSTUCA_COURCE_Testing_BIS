# Задание на лабораторные работы

## Темы лабораторных работ согласно рабочей программе

* Лабораторная работа №1: "Разработка документации (Техническое задание, пояснительная записка)".
* Лабораторная работа №2: "Написание программы, юнит-тесты (функций) и смоук-тест комплекса".
* Лабораторная работа №3: "Комплексное тестирование".

## Задание

- Разработайте техническое задание (см. ГОСТ 19) для программы обработки текстовых файлов.
- Составьте пояснительную записку (см. ГОСТ 19), включающую:
    - Цели и задачи программы
    - Ожидаемые входные данные и результаты
    - Предполагаемую структуру программы (модули/функции)
    - Требования к характеристикам программы
- Включите план тестирования с описанием видов тестов и ожидаемых результатов.

Результатом работы должен быть текстовый документ с техническим заданием и пояснительной запиской.


Задание:
- Реализуйте программу согласно техническому заданию из ЛР №1.
- Напишите юнит-тесты для основных функций программы.
- Проведите смоук-тест программы на тестовом файле.

Результатом работы должны быть:
- Исходный код программы
- Функция unit_tests() с юнит-тестами
- Отчёт о результатах смоук-теста

Задание:
- Проведите комплексное тестирование программы, написанной в ЛР №2.
- Включите следующие виды тестирования:
    - Функциональное тестирование
    - Нефункциональное тестирование
    - Тестирование производительности
    - Тестирование удобства пользования
    - Тестирование безопасности
- Напишите отчёт о тестировании, включающий:
    - Описание проведённых тестов
    - Результаты тестирования
    - Анализ производительности
    - Выводы о соответствии программы требованиям

Если обнаружены ошибки, исправьте их и повторите тестирование.

Варианты программ для использования в лабораторных работах (ну или что-нибудь своё):

1. Программа анализа текстовых документов:
   - Подсчёт строк, слов и символов
   - Поиск самого длинного слова
   - Анализ частоты букв

2. Программа подсчёта статистики продаж:
   - Обработка CSV-файлов с данными о продажах
   - Расчёт общих сумм продаж
   - Поиск самых популярных товаров
   - Анализ динамики продаж

3. Программа анализа лог-файлов:
   - Подсчёт количества запросов
   - Определение наиболее часто встречающихся IP-адресов
   - Анализ времени ответа сервера
   - Поиск ошибок в логах

4. Программа обработки данных опросов:
   - Подсчёт ответов на вопросы
   - Определение наиболее популярных вариантов ответов
   - Анализ демографических данных
   - Расчёт процентного соотношения ответов
